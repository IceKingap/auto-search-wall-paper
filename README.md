# Auto Wallpaper

Android用のライブ壁紙アプリ。画面ON/OFFやホーム復帰時に壁紙を自動切り替えし、クロスフェードとズームアウトアニメーションで表示します。

## 機能

### 壁紙表示
- **クロスフェード切り替え** - 画面ON時に次の壁紙へ滑らかにフェード
- **ズームアウトアニメーション** - 壁紙表示時に画面いっぱいから画像全体表示へ引いていくアニメーション（減速補間）
- **表示モード** - COVER（画面全体を覆う）/ FIT（画像全体を表示）
- **スティックパン** - ホーム画面で指を引っ張ると画像がその方向に動き続ける（速度・方向反転を設定可能）
- **余白の色補完** - 黒い余白を画像の色合いで埋める（ぼかし幅も設定可能）

### 画像ソース
- **ローカルアルバム** - 端末内の画像をアルバム管理
- **Safebooru・danbooru連携** - タグ検索でオンライン画像をダウンロード・キャッシュ
- **自動ダウンロード** - 保存した検索クエリをローテーションして自動補充（消費→補充モデル）

### クロップ
- **カスタムクロップ** - ドラッグ可能なハンドルで範囲指定（三分割グリッド表示）
- **左右分割** - 横長画像を任意の位置で分割し交互に表示

### オーバーレイ
- **ステータスバーオーバーレイ** - 白い画像でも時刻が見えるよう半透明バーを表示（高さ調整可能）
- **ロック画面オーバーレイ** - ロック画面のみに表示される角丸の半透明四角形（位置・サイズ・不透明度を設定可能、複数追加可）

### その他
- 選択順序: ランダム / 順番 / お気に入り優先
- ホーム復帰時の壁紙切り替えON/OFF
- Wi-Fiのみダウンロード
- 壁紙履歴（サムネイル付き）
- お気に入り管理
